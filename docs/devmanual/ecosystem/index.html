Open Source Projects

Community maintained open source projects that extend Otoroshi with new capabilities. Either with simple plugins or more complex extensions.

otoroshi-llm-extension

Turn Otoroshi into an AI Gateway. Connect, setup, secure and seamlessly manage AI models using a Universal/OpenAI compatible API.

AIAI GatewayLLM

otoroshi-waf-extension

A Web Application Firewall (WAF) extension for Otoroshi with ModSecurity SecLang and OWASP CRS.

SecurityWAF

otoroshi-biscuit-studio

Biscuit studio Extension for Otoroshi. Create, manage and validate Eclipse Biscuit authorization tokens.

SecurityAuthorizationstoken

otoroshi-plugin-dynamic-js-modules

Execute WASM plugins written in JavaScript without any compilation phase.

PluginWASM

otoroshictl

A CLI to manage your Otoroshi clusters with style. Import, export, sync and automate operations.

CLITools

otoroshi-plugin-webhook-validator

Webhook validation plugin for Otoroshi. Validate incoming webhook signatures and payloads.

PluginWebhook

otoroshi-plugin-mailer

An Otoroshi plugin to send emails through an asynchronous REST API.

PluginEmail

otoroshi-plugin-couchbase

Couchbase storage backend plugin for Otoroshi.

PluginStorage

otoroshi-plugin-spiffe

SPIFFE/SPIRE integration plugin for Otoroshi. Manage workload identity with zero-trust networking.

SecurityZero Trust

otoroshi-plugin-moesif

Moesif API analytics integration plugin for Otoroshi.

PluginAnalytics

otoroshi-plugin-curity-phantom-token

Curity Phantom Token integration plugin for Otoroshi.

PluginAuth

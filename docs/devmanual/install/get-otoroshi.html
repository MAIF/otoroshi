Get Otoroshi

All release can be bound on the releases page of the repository.

From zip

# Download the latest version
wget https://github.com/MAIF/otoroshi/releases/download/v17.4.0-dev/otoroshi-17.4.0-dev.zip
unzip ./otoroshi-17.4.0-dev.zip
cd otoroshi-17.4.0-dev

From jar file

# Download the latest version
wget https://github.com/MAIF/otoroshi/releases/download/v17.4.0-dev/otoroshi.jar

From Docker

# Download the latest version
docker pull maif/otoroshi:17.4.0-dev-jdk11

From Sources

To build Otoroshi from sources, just go to the dev documentation
